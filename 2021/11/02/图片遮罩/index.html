hover图片显示遮罩

2021-11-02

参考链接：https://blog.csdn.net/Asuna_Yu/article/details/80259194

为图片及遮罩层添加样式

图片：relative

遮罩层：absolute

使两者样式重合。

鼠标不在图片上时，遮罩层不显示 .mask { opacity: 0; } 。

使用hover

改变透明度，使遮罩层显示。
